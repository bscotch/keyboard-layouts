Additional features

AltGr: This layout treats right Alt key as Control+Alt.

Directionality: The layout inserts Left-to-Right Marker (LRM) on Left Shift+Backspace and Right-to-Left Marker (RLM) on Right Shift+Backspace. Furthermore, Left Shift+Ctrl and Right Shift+Ctrl change the directionality of a paragraph, or a text box to LTR and RTL respectively.
